# GlassblowersToolchest
An Android app for Glassblowers


FEATURES
  Marble Calculator
    Calculate the amount of glass rod needed to make a marble
  Anneal-o-Matic
    Generate a kiln schedule based on the thickness of glass to be annealed
  Polariscope
    Check how much stress is trapped in your glass
  COA Generator
    Create a Certificate of Authenticity for your work
